## About The App/ How to use:

This is a simple API app that you can get the best astrophotographic photo of NASA resources every day! You can select the date you want from the clender and click the 'Ready?' button. 
Voila you will get the coolest picture of the day and also the explanation of it! 

## How It's Made:

This app was my first ever attempt using APIs. I love NASA and astronogy so when I learned about API and they found out NASA has an open APIs that we can use for free? I have to try this too!

## Tech Used:
HTML, CSS, JavaScript
